# Changelog Rules

## 核心原则

### SemVer 版本号
```
主版本.次版本.修订号
MAJOR: 不兼容变更
MINOR: 向后兼容新增
PATCH: 向后兼容修复
```

### CHANGELOG 类型
| Type | 说明 |
|------|------|
| Added | 新功能 |
| Changed | 变更功能 |
| Fixed | Bug 修复 |
| Removed | 移除功能 |
| Deprecated | 废弃功能 |

### 自检
- [ ] Type 选择正确
- [ ] Breaking Changes 已标注
- [ ] 版本号符合 SemVer
