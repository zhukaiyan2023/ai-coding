# Code Quality Rules

## 评分体系

| 加分项 | +分 | 扣分项 | -分 |
|--------|-----|--------|-----|
| JSDoc/TSDoc | +5 | TODO/FIXME | -5 |
| TypeScript Types | +3 | console.log | -3 |
| Error Handling | +5 | debugger | -10 |
| Tests | +3 | Hardcoded | -2 |

### 评分等级
| 分数 | 等级 |
|------|------|
| ≥ 10 | A |
| 5-9 | B |
| 0-4 | C |
| < 0 | D |

## 复杂度阈值
| 指标 | 阈值 |
|------|------|
| 函数行数 | ≤ 50 |
| 文件行数 | ≤ 300 |
| 圈复杂度 | ≤ 10 |

## 自检
- [ ] 质量评分 ≥ 0
- [ ] 函数 ≤ 50 行
- [ ] 有错误处理
- [ ] 有类型标注（TS）
